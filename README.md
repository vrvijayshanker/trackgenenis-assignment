STEPS TO RUN THE PROJECT LOCALLY
--------------------------------------
1. Go to https://github.com/vrvijayshanker/trackgenenis-assignment
2. Download the project as ZIP file
3. Unzip into desired directory
4. Go to the Project directory and Open it with VSCode (Or go to directory within VSCODE Terminal)
5. Make sure Node.js is installed on your computer
6. In the terminal, type the command: 'npm install' (This installs all the packages needed to run the project)
7. Then enter command: 'npm start' in the terminal
8. Project will start running locally in the Port 3000
9. The default browser will automatically launch the application.
10 If not, enter http://localhost:3000/ into your favourite browser's address bar and enter.
